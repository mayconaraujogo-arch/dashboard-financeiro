# Dashboard Financeiro Maycon

Site financeiro local em tema roxo, pronto para GitHub Pages.

## Como testar no PC
Abra o arquivo `index.html` no navegador.

## Como publicar no GitHub Pages
1. Crie um repositório no GitHub chamado `dashboard-financeiro`.
2. Envie todos os arquivos desta pasta para o repositório.
3. Vá em `Settings > Pages`.
4. Em `Build and deployment`, selecione:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Clique em Save.
6. O GitHub vai gerar um link como:
   `https://seuusuario.github.io/dashboard-financeiro/`

## Importante
Os dados são salvos no navegador usando localStorage.
Para levar os dados para outro aparelho, use:
- Backup > Exportar backup JSON
- No outro aparelho: Backup > Importar backup
